# Как запустить проект

## Запуск сервера

### Вариант 1: Через batch файл
Дважды кликните на файл `START_SERVER.bat` в корне проекта.

### Вариант 2: Через командную строку
```bash
cd c:\projects\einvestor-laravel
php artisan serve --port=3000
```

## URL проекта

После запуска сервера откройте в браузере:

**http://localhost:3000**

## Если порт занят

Если порт 3000 занят, можно использовать другой порт:

```bash
php artisan serve --port=3001
```

Или освободить порт 3000:

```powershell
# Найти процесс на порту 3000
netstat -ano | findstr ":3000"

# Остановить процесс (замените PID на реальный ID процесса)
taskkill /PID <PID> /F
```

## Проверка работы

После запуска сервера вы должны увидеть:
```
INFO  Server running on [http://127.0.0.1:3000]
```

## Остановка сервера

Нажмите `Ctrl+C` в окне терминала, где запущен сервер.
